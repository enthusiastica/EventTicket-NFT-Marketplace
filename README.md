# First Ink

Created from a Next.js TypeScript [starter](https://github.com/redimpulz/nextjs-typescript-starter).

## Tech stack

- Next.js
- TypeScript
- Supabase
- Vercel

## Getting started

1. Install dependencies:

`yarn`

2. Start the development server:

`yarn dev`

3. Set up environment variables.
